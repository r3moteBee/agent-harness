DEPLOYMENT INSTRUCTIONS
================================================================================

1. DOCKER BUILD & RUN:
   docker build -t pantheon-frontend .
   docker run -p 80:80 pantheon-frontend

2. REVERSE PROXY:
   The frontend runs on port 80 in Docker (handled by nginx)
   Configure your infrastructure to route to this container

3. BACKEND CONNECTION:
   Update API_URL in vite.config.js to point to your backend
   Default: http://localhost:8000

4. ENVIRONMENT:
   Frontend requires backend API running on configured URL
   WebSocket support required for chat streaming
